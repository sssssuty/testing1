A digital platform to learn Simplified Chinese and Mandarin by decoding the system of forms. With detail information in character stroke structure, sequence in writing, evolution history, and pronunciation, the platform contains knowledge to benefit users with all levels.


Main function #1: foundation
Page foundation is where users can find all definitions, proper nouns, symbols, and history, making sure all users and the website is calling the same thing the same name.

Main function #2: browse
Page browse is where users can find all 6000+ Simplified Chinese characters.

Main function #3: chapters
Page chapter is where the first magic happens. Users can find groups of characters and learn them by the decoded rules in shape.

Main function #4: playground
Page playground is where users can play around with the input characters, by pulling around the corner of each stroke.

Main function #5: search
Depending on the length of input into search bar, the result varies. If the user types in short phrase or sentence, the result will look like the sample page below. If the user types in a single character, then the result will be the detail page of the input character.
